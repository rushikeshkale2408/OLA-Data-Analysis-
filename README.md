# OLA-Data-Analysis-
Domain

Transportation / Ride Sharing

Tools Used

SQL, Power BI, Excel

Dataset

The dataset contains booking details such as ride distance, fare, pickup location, and time.

KPIs

- Total Bookings
- Total Revenue
- Average Ride Distance
- Cancellation Rate

Insights

- Peak demand observed during evening hours
- Higher bookings on weekends
- Certain locations show consistently high demand

Conclusion

This analysis helps in understanding customer behavior and improving driver allocation and pricing strategies
